# Installing R and RStudio

## Concepts

- **Install R first, then RStudio.** RStudio needs an existing R installation and auto-detects it during setup (see [[introduction-to-r-and-rstudio]]).
- **Install R (Windows):** go to r-project.org > "download R" (under Getting Started) > pick a CRAN mirror > "Download R for Windows" > "install R for the first time" > download the version's `.exe`, run it, follow the prompts.
- **Install R (Mac):** r-project.org > "download R" > pick a CRAN mirror > "Download R for (Mac) OS X" > download the latest `.pkg` under "Files" > double-click and follow prompts.
- **Install RStudio (both OSes):** rstudio.com > "Download RStudio" > under RStudio Desktop click "Download" > pick the build for your OS (Windows `.exe`, or Mac `.dmg` - drag into Applications) > run/install.
- A **CRAN mirror** is one of many geographically distributed servers hosting the same R download files; pick a nearby one.

## Code examples

None - installation is done through the websites and OS installers, with no console commands.

## Gotchas

- Order matters: R before RStudio.
- The lesson's screenshots pin R 3.4.2; install the current release instead - the steps are otherwise unchanged.
- The lesson points to the official R FAQ and RStudio FAQ for install troubleshooting.
