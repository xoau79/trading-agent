# Skills

One markdown file per recurring operational task around this repo. Each file has the same four
sections: when to use it, the exact steps in order, a real example of good output, and the
mistakes to avoid (each traceable to an actual correction in this repo's history). Written so
a model with no prior context on this project can execute the task cold.

| File | Task |
|---|---|
| [`review-and-merge-prs.md`](review-and-merge-prs.md) | Reviewing, testing, and merging a pull request |
| [`monitor-live-sessions.md`](monitor-live-sessions.md) | Watching an Asian/New York session live via dashboard/Discord/logs |
| [`review-journal-and-weekly-stats.md`](review-journal-and-weekly-stats.md) | Reading trade archives, session reviews, and the weekly stats/tuning-gate report |
| [`run-backtests-and-replays.md`](run-backtests-and-replays.md) | Running `bot.py --backtest` replays and `backtest.py` multi-session backtests |
| [`decide-agent-suggestions.md`](decide-agent-suggestions.md) | Approving or rejecting a pending agent-proposed parameter change |
| [`check-feed-health.md`](check-feed-health.md) | Running the smoke test and the broker/Yahoo feed parity check |
| [`verify-demo-broker.md`](verify-demo-broker.md) | Setting up and verifying a cTrader/MT5 demo account before trading it |
